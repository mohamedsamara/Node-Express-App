# Node Express App

## Install

Some basic Git commands are:

```
$ git clone https://github.com/mohamedsamara/Node-Express-App.git
$ cd project
$ npm install

```

## Start

npm start

## Languages & tools

- [Node](https://nodejs.org/en/)

- [Express](https://expressjs.com/)

- [Pug](https://pugjs.org/api/getting-started.html)
